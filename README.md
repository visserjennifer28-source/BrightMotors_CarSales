# Bright Motors Car Sales Analysis

## Overview
This project analyzes car sales data to uncover insights that can help improve sales performance, optimize inventory, and guide strategic decision-making.

## Tools Used
- SQL
- Excel / Power BI
- PowerPoint

## Key Insights
- High-end car brands generate the most revenue
- Lower mileage vehicles sell at higher prices
- SUVs and Sedans dominate sales
- Regional performance varies significantly

## Recommendations
- Focus on high-revenue vehicles
- Expand in top-performing regions
- Improve pricing strategies for older vehicles
